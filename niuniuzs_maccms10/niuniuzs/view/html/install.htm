欢迎使用牛牛助手-影视的好帮手

安装说明

0、数组字符：输入任意内容系统会进行API密钥设置
1、采集接口：【自动添加】所有推荐资源到采集接口 【手动添加】 根据需求手动添加
2、 播 放 器：【自动添加】所有推荐资源的播放器 【手动添加】 根据需求手动添加
3、定时任务：【自动添加】所有推荐资源到定时任务 【手动添加】 根据需求手动添加
4、分类导入：【不导入】保持现有分类 【导入影视【x】分类】 不会覆盖掉现有的分类 只会增加没有的
5、分类数据：【不清空原分类】保留现有分类数据 【清空原分类】 清空所有分类数据 需要配合 分类导入使用
6、会员权限：【自动设置】自动设置添加的分类 会员组查看权限 【手动设置】 后台自己修改权限
7、自动分类：【自动分类】自动绑定现有分类 【手动分类】 手动绑定现有分类
